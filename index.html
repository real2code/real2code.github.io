Real2Code: Reconstruct Articulated Objects
via Code Generation

Mandi Zhao1 , Yijia Weng1, Dominik Bauer 2, Shuran Song 1,2

1 Stanford University   2 Columbia University

Abstract

We propose Real2Code, a novel method for reconstructing articulated objects via code generation, leveraging pre-trained large language models (LLMs). Real2Code takes visual observations as input, and performs both part-level geometry reconstruction and joint prediction. We evaluate our method on an extensive set of objects with varying level of kinematic complexity. Whereas prior methods fail at objects with more than three articulated parts, Real2Code reliably reconstructs articulated objects with up to 10 parts.
